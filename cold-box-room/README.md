# cold-box-room

Python package for the **Cold-box-room** autonomous DFIR hallway agent.

**Full submission README (hackathon components 1–8):** [`../README.md`](../README.md)

```bash
pip install -e ".[dev]"
```

```bash
cold-box-room-hallway --help

```
